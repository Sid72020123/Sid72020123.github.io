Learn HTML

HTML
HTML stands for HyperText Markup Language. HTML is used to create good looking and interesting webpages and websites like this one! We can also use CSS and JavaScript to make a more good-looking and responsive webpage! The various functions of HTML are written between < and >.
Following is a simple HTML program:

<Html>
<Head>
<Title>
Sample Webpage
</Title>
</Head>

<Body>
Hi!
</Body>
</Html>

Explanation:

  1. The first line <Html> states the start of a HTML document.
  2. The second line <Head> states the start of Head.
  3. The third line <Title> states the start of Title. The title is displayed on the browser's tab.
  4. The fourth line Sample Webpage just shows what the title should be.
  5. The fifth line </Title> shows that it is the end of the Title text.
  6. The sixth line </Head> states the end of <Head> tag.
  7. The seventh line is a empty line. There should be line breaks in code to make it clear.
  8. The eighth line <Body> states the start of Body tag. The contents inside the <Body> and </Body> are shown in the webpage.
  9. The nineth line Hi! is the text that should appear in the webpage.
  10. The tenth line </Body> states the end of Body tag.
  11. The last line </Html> states the end of a HTML document.

Important Points:

Webpage: Webpages are the individual pages of a website. Webpages contain many things like information, music, games, etc.
Website: Website is a collection of webpages on a certain topic.
