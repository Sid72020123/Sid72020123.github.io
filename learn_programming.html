HTML, CSS and Javascript

Hypertext Markup Language(HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets(CSS) and scripting languages such as JavaScript.

Python Programming

Python is an interpreted, high-level and general-purpose programming language. Python is very simple to learn for beginners and is very useful language.

Python is used to: (Source: w3schools.com)

Java Programming

Java is a class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.
Java is a popular programming language, created in 1995. It is owned by Oracle, and more than 3 billion devices run Java.

Java is used to: (Source: w3schools.com)
  • Mobile applications (specially Android apps)
  • Desktop applications
  • Web applications
  • Web servers and application servers
  • Games
  • Database connection
  • And much, much more!
Start Learning Java now!
